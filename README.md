# Test challenge for Incust company

**Instructions:**

- Clone this repository to your desired location
- `cd` into repo folder
- Run docker-compose:
```
sudo docker-compose up
```
- API is now working on your machine. You should now be able to access Swagger UI by visiting [https://localhost:5000/ui/](https://localhost:5000/ui/) in your browser


Source code is covered by tests. In order to run them, you can enter the running `web` container and execute `pytest` command
